# Math Object
Es uno de los objetos *built-in* de JavaScript.

Si queremos ver todos los métodos y propiedades disponibles podemos `console.log(Math)`
```jsx
Math.random() //número random entre 0 y 1 (exclusivo).
Math.floor(5.99) //5
Math.ceiling(5.01) //6
Math.trunc(5.2) //5

Math.round(5.4) //5
Math.round(5.5) //6

Math.PI
```
